# 第十章：实战项目1 - 我的第一个网页

> 📖 **难度等级**: ⭐⭐ (简单)
> ⏰ **预计完成时间**: 30-60分钟
> 🎯 **学习目标**: 从零开始，用AI完成一个完整的个人网页

---

## 🎯 项目介绍

### 我们要做什么

创建一个漂亮的**个人介绍网页**，包含：
- 📸 个人头像
- 👤 个人信息（姓名、职业、简介）
- 📧 联系方式
- 🎨 现代化的设计
- 📱 响应式布局（手机也能看）

### 学习目标

完成这个项目，你将学会：
- ✅ 用AI创建项目结构
- ✅ 理解HTML基本标签
- ✅ 了解CSS基础样式
- ✅ 掌握响应式设计概念
- ✅ 学会如何修改和改进页面

---

## 🚀 项目准备

### 准备清单

开始之前，请确保：

- [ ] 你已经安装并熟悉了AI工具（Trae/Cursor/Claude Code）
- [ ] 选择了你喜欢的配色方案（稍后可以用）
- [ ] 准备好个人介绍的信息：
  - 姓名
  - 职业/身份
  - 简短介绍（1-2句话）
  - 联系邮箱（可选）

### 项目规划

```
项目结构：
personal-website/
├── index.html      # 主页面
├── style.css       # 样式文件
└── README.md      # 项目说明（可选）

页面布局：
┌────────────────────────────────────┐
│          顶部导航栏                │
├────────────────────────────────────┤
│                                  │
│        [头像]                     │
│         姓名                      │
│         职业                      │
│       个人简介                    │
│                                  │
│       [联系我]                   │
│                                  │
└────────────────────────────────────┘
```

## 📝 步骤1：创建项目

### 使用Trae创建

```
操作：

1. 打开Trae

2. 和AI对话：
你：帮我创建一个个人介绍网页项目

3. AI会响应：
AI: 好的！我来帮你创建...

正在创建项目...
✓ personal-website 文件夹
✓ index.html - 主页面
✓ style.css - 样式文件
✓ 基础结构已搭建完成

4. 查看生成的文件
在左侧面板中可以看到创建的文件
```

### 使用Cursor创建

```
操作：

1. 打开Cursor

2. 打开Chat模式（Ctrl/Cmd + L）

3. 发送请求：
你：创建一个个人介绍网页，
包含头像、姓名、职业、简介和联系方式

4. AI生成代码后，创建新文件夹：
   - File → New Folder → 命名为 personal-website

5. 创建文件：
   - index.html
   - style.css

6. 复制AI生成的代码到相应文件
```

### 使用Claude Code创建

```
操作：

1. 打开终端，运行claude-code

2. 告诉AI：
你：创建一个个人介绍网页项目

3. AI自动创建：
AI: 正在创建项目...
✓ 创建项目文件夹
✓ 生成 index.html
✓ 生成 style.css

4. 查看文件：
用代码编辑器打开生成的文件查看
```

## 📝 步骤2：生成基础页面

### 给AI的详细需求

```
把这段话发给AI：

帮我创建一个个人介绍网页，要求如下：

页面结构：
- 顶部导航栏，显示我的名字
- 主要内容区，居中显示
- 底部版权信息

主要内容包含：
1. 头像（使用圆形占位图标）
2. 姓名：[你的名字]
3. 职业：[你的职业]
4. 简介文本：[你的简介]
5. 联系邮箱按钮

样式要求：
- 现代简洁的设计风格
- 使用渐变背景（蓝色到紫色）
- 白色卡片式布局
- 圆角和阴影效果
- 响应式设计（手机和电脑都能看）

技术要求：
- 使用HTML5
- 使用CSS3
- 代码清晰易读
- 添加必要的注释
```

### AI生成的代码示例

AI会生成类似这样的代码：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>个人介绍 - [你的名字]</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <nav class="navbar">
        <div class="nav-content">
            <h1>[你的名字]</h1>
        </div>
    </nav>

    <main class="container">
        <div class="card">
            <div class="avatar">👤</div>
            <h2 class="name">[你的名字]</h2>
            <p class="title">[你的职业]</p>
            <p class="bio">[你的简介]</p>
            <a href="mailto:your@email.com" class="btn">
                联系我
            </a>
        </div>
    </main>

    <footer class="footer">
        <p>&copy; 2026 [你的名字]. All rights reserved.</p>
    </footer>
</body>
</html>
```


## 📝 步骤3：预览和调整

### 预览效果

```
方法1：在浏览器中打开
- 双击 index.html 文件
- 或右键 → Open with → 浏览器

方法2：使用工具的预览功能

- Cursor: 右键 → Open in Default Browser
- Claude Code: 用浏览器打开文件
```

### 调整和改进

第一次生成的页面可能不是完美的，继续改进：

```
改进1：修改个人信息
你：把名字改成[你的真实名字]，职业改成[你的真实职业]

改进2：调整颜色
你：背景颜色太深了，请调整成浅一点的渐变

改进3：添加动画
你：给头像添加一个缓慢的旋转动画效果

改进4：添加社交链接
你：在联系方式下面添加GitHub和Twitter的图标链接

改进5：优化手机显示
你：在手机上卡片有点挤，请调整手机上的样式
```

### 示例：添加头像旋转动画

```

和AI说：
你：给头像添加一个缓慢的旋转动画，每10秒转一圈

AI会添加CSS动画：

@keyframes rotate {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
}

.avatar {
    /* ... 其他样式 ... */
    animation: rotate 10s linear infinite;
}

现在头像会慢慢旋转了！
```

## 📱 步骤6：响应式优化

### 测试不同屏幕

```
在浏览器中测试：

1. 打开开发者工具（F12）
2. 切换设备模拟：
   - 手机竖屏
   - 手机横屏
   - 平板
   - 桌面

3. 检查是否有问题：
   - 文字是否太小
   - 卡片是否太挤
   - 按钮是否容易点击

有问题就告诉AI修复！
```

### 常见响应式问题

```
问题1：手机上文字太小
你：在手机上文字太小了，
请调整移动端的字体大小

问题2：卡片在手机上太挤
你：手机上卡片和边界的距离太小，
请增加内边距

问题3：导航栏在手机上不合适
你：手机上的导航栏样式不太好，
请专门为手机优化一下
```
