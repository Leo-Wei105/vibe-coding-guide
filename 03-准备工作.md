## 💻 电脑要求

### 最低配置（能用）：
- 操作系统：Windows 10/11、macOS 12+、Linux
- 内存：4GB或更多
- 存储空间：10GB可用空间
- 网络：稳定的互联网连接

### 推荐配置（更好体验）：
- 操作系统：Windows 11、macOS 14+、Linux（最新版）
- 内存：8GB或更多
- 存储：20GB可用空间
- 网络：高速宽带连接

> 💡 **小贴士**：几乎任何现代电脑都能用来做vibe coding，不需要特别的硬件！

---

## 🌐 网络准备

### 为什么需要网络？

vibe coding的核心是AI，AI需要联网才能工作。你需要：

✅ **稳定的网络连接**
- 不需要超高速，但要稳定
- 最好不是经常断网的移动网络

✅ **能访问AI服务**
- Claude（推荐）
- OpenAI/ChatGPT
- 或其他AI服务

### 网络测试方法：

```bash
# Windows：在命令行（CMD）中输入
ping claude.com

# Mac/Linux：在终端中输入
ping claude.com
```

如果能ping通，说明网络没问题！

## 📦 第一步：安装基础软件

### 必须安装的软件：

#### 1. 代码编辑器

虽然AI会帮你写代码，但你还是需要一个地方来查看和编辑代码。

**推荐选择：**

| 编辑器 | 难度 | 适合场景 |
|--------|------|---------|
| VS Code | ⭐⭐⭐ | 最通用的选择 |
| Cursor | ⭐ | 内置AI，最方便 |
| Trae | ⭐ | 最简单，中文友好 |

**VS Code安装步骤：**

1. 访问官网：https://code.visualstudio.com/
2. 下载适合你操作系统的版本
3. 运行安装程序
4. 一路"下一步"完成安装

#### 2. 浏览器（你可能已经有了）

推荐使用：
- Chrome
- Edge（Windows自带）
- Safari（Mac自带）

#### 3. Git（可选但推荐）

Git是用来管理代码版本的工具，之后项目复杂了会用到。

- 访问：https://git-scm.com/
- 下载安装
- 安装时选择默认选项即可

## 🎨 第二步：选择你的第一个工具

### 选项A：Trae（最简单，推荐初学者）

**为什么选择Trae？**
- ✅ 完全免费
- ✅ 中文界面
- ✅ 最容易上手
- ✅ 内置所有需要的功能

**安装步骤：**
1. 访问：https://trae.ai/
2. 点击"下载"
3. 运行安装程序
4. 打开应用，创建账号或登录
5. 完成！

### 选项B：Cursor（推荐进阶用户）

**为什么选择Cursor？**
- ✅ 基于VS Code，界面熟悉
- ✅ 强大的AI功能
- ✅ 有免费额度
- ✅ 社区活跃

**安装步骤：**
1. 访问：https://cursor.sh/
2. 点击"Download for Windows/Mac/Linux"
3. 运行安装程序
4. 打开应用，创建账号
5. 完成初始设置

### 选项C：Claude Code（推荐已有Claude用户）

**为什么选择Claude Code？**
- ✅ Anthropic官方工具
- ✅ 与Claude深度集成
- ✅ 强大的代码理解能力
- ✅ 支持多种工作方式

**安装步骤：**
1. 打开终端/命令行
2. 输入安装命令：

```bash
# Mac - 使用Homebrew
brew install claude-code

# Windows - 使用WinGet
winget install Claude.ClaudeCode

# 或者使用npm（需要先安装Node.js）
npm install -g @anthropic-ai/claude-code
```

3. 安装完成后，运行：
```bash
claude-code
```

4. 按提示登录你的Claude账号

## 🗂️ 第四步：创建你的第一个项目

### 创建项目文件夹：

在你的电脑上，找一个合适的位置创建项目文件夹：

```
C:\Users\你的名字\my-projects\    # Windows
~/my-projects/                     # Mac/Linux
```

### 创建第一个项目：

```
my-projects/
├── hello-world/          # 第一个项目
│   ├── index.html        # 网页文件
│   └── README.md         # 项目说明
```

**在Trae/Cursor中：**
1. 打开工具
2. 选择"新建项目"
3. 选择"HTML/网页"模板
4. 给项目命名（如：hello-world）
5. 创建！

## 🔧 常见问题解决

### 问题1：安装失败

**症状**：安装过程中出错

**解决方案**：
1. 检查网络连接
2. 尝试重新下载安装包
3. Windows用户：以管理员身份运行
4. 检查是否有杀毒软件拦截

### 问题2：账号无法注册

**症状**：注册时收不到验证码

**解决方案**：
1. 检查手机号格式是否正确
2. 尝试更换浏览器
3. 使用其他手机号
4. 联系客服寻求帮助

### 问题3：AI不回复

**症状**：发送消息后，AI没有回复

**解决方案**：
1. 检查网络连接
2. 检查账号额度是否用完
3. 查看是否有错误提示
4. 重启工具重试

### 问题4：工具卡住或闪退

**症状**：工具突然不响应或关闭

**解决方案**：
1. 关闭其他占内存的程序
2. 重启工具
3. 检查工具是否需要更新
4. 卸载后重新安装

## ⚠️ 避坑指南：安装常见问题

### ❌ 问题1：安装后打不开

**症状**：双击图标没反应，或者闪退

**解决方法**：
```
Windows用户：
1. 右键点击图标 → "以管理员身份运行"
2. 检查杀毒软件是否拦截
3. 尝试重新下载安装

Mac用户：
1. 系统偏好设置 → 安全性与隐私 → 允许打开
2. 右键点击应用 → 打开
```

### ❌ 问题2：登录不成功

**症状**：一直显示登录中，或者收不到验证码

**解决方法**：
```
1. 检查网络连接是否正常
2. 尝试更换浏览器登录
3. 检查手机号/邮箱是否正确
4. 稍后再试（服务器可能繁忙）
```

### ❌ 问题3：AI不回复

**症状**：发送消息后，一直显示加载中

**解决方法**：
```
1. 检查网络连接
2. 检查账号额度是否用完
3. 尝试切换AI模型（如果工具支持）
4. 重启工具
5. 如果是Claude/ChatGPT，检查是否需要特殊网络
```

### ❌ 问题4：不知道选哪个工具

**快速决策树**：
```
你会英文吗？
├── 不太会 → 选 Trae（中文最好）
└── 会一点
    ├── 想免费 → 选 Trae
    └── 愿意付费
        ├── 喜欢图形界面 → 选 Cursor
        └── 喜欢命令行 → 选 Claude Code
```

### 💡 遇到问题的万能方法

如果上面都解决不了，试试这个魔法咒语（用任何能用的AI）：

```prompt
我在安装[工具名]时遇到了问题。

我的情况：
- 操作系统：[Windows/Mac/Linux]
- 错误现象：[描述你看到的]
- 已经尝试：[你试过什么方法]

请帮我分析可能的原因和解决方法。
```
