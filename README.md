## 📖 这是什么？

这是一份面向**零基础用户**的 Vibe Coding 完全教程。

**Vibe Coding** 是一种全新的编程方式：
- 🗣️ 用**自然语言**告诉 AI 你想做什么
- 🤖 AI 帮你**自动生成代码**
- ✨ 你只需要**检查和调整**

**目标读者**：
- 完全没有编程经验的小白
- 想快速实现想法的创业者
- 对 AI 编程工具感兴趣的任何人
