## 📝 第一次体验：Hello World

### 为什么是Hello World？

"Hello World"是所有编程教程的第一个练习。它的意义是：
- 最简单的代码
- 验证环境是否正常
- 给你成就感："我能写代码了！"

### 任务1：让AI创建Hello World

#### 在Trae中操作：

```
步骤1：打开Trae

步骤2：创建新项目
→ 点击 "新建项目"
→ 选择 "网页项目"
→ 命名为 "hello-world"

步骤3：和AI对话
你：帮我创建一个Hello World网页，显示"你好，世界！"

AI：好的！我来帮你创建...

正在生成...
✓ index.html - 网页文件
✓ 已添加文字"你好，世界！"
✓ 已添加简单样式

步骤4：预览
点击预览按钮
→ 你会看到页面显示：你好，世界！

恭喜！你成功完成了第一个AI编程任务！🎉
```

#### 在Cursor中操作：

```
步骤1：打开Cursor

步骤2：打开Chat模式（Ctrl/Cmd + L）

步骤3：发送请求
你：创建一个Hello World网页，显示"你好，世界！"

步骤4：AI生成代码
AI: 好的，我来创建...
（生成完整的HTML代码）

步骤5：复制代码并保存
复制AI生成的代码
创建 index.html
粘贴代码

步骤6：预览
右键 → Open in Default Browser
→ 看到页面显示：你好，世界！

完成！
```

#### 在Claude Code中操作：

```
步骤1：打开终端，运行claude-code

步骤2：告诉AI你的需求
你：创建一个Hello World网页，显示"你好，世界！"

步骤3：AI自动创建
AI: 正在创建文件...
✓ 已创建 index.html
✓ 已写入HTML代码

步骤4：查看文件

```

```
✓ 添加了CSS样式

✓ 文字居中显示
✓ 添加了淡入动画

预览效果：
页面现在有一个漂亮的渐变背景，
文字从中间优雅地显示出来！
```

### AI添加的代码示例：

```html
<!DOCTYPE html>
<html>
<head>
    <style>
        body {
            margin: 0;
            min-height: 100vh;
            background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
            display: flex;
            justify-content: center;
            align-items: center;
            font-family: Arial, sans-serif;
        }

        h1 {
            color: white;
            font-size: 48px;
            animation: fadeIn 1s ease-in;
        }

        @keyframes fadeIn {
            from { opacity: 0; }
            to { opacity: 1; }
        }
    </style>
</head>
<body>
    <h1>你好，世界！</h1>
</body>
</html>
```

## 📝 常用操作流程

### 流程1：创建新功能

```
步骤1：描述需求
你：我想给页面添加一个按钮

步骤2：AI生成代码
AI：好的，添加一个按钮...

步骤3：查看和运行
运行代码，看看效果

步骤4：调整改进
你：按钮太小了，把它变大一点

AI：已调整按钮大小...

步骤5：继续改进
你：给按钮添加点击效果

AI：已添加点击动画...

直到满意为止！
```

### 流程2：修复错误

```
情况：代码运行出错了

步骤1：复制错误信息
TypeError: Cannot read property 'length' of undefined

步骤2：告诉AI错误
你：我的代码报错了：
TypeError: Cannot read property 'length' of undefined

步骤3：AI分析原因
AI：这个错误是因为...
- 变量可能是undefined
- 没有检查是否存在就直接访问属性

步骤4：AI提供修复
AI：建议这样修复：
if (array && array.length > 0) {
    // 安全访问
}

步骤5：应用修复
把修复代码应用到项目中

步骤6：验证
再次运行，确认问题解决
```

### 流程3：理解代码

```
情况：AI生成的代码，你不太懂

步骤1：选中代码
选中你看不懂的部分

步骤2：让AI解释
你：这段代码是什么意思？

步骤3：AI详细解释
AI：这段代码实现了一个功能...

具体说明：
- 每行代码的作用
- 为什么要这样写
- 有什么注意事项

步骤4：确认理解
你：所以这个函数是用来做X的，对吗？

AI：是的！正确！

步骤5：继续学习
如果还不懂，继续问！
```

## 💡 基础操作技巧

### 技巧1：如何清晰地描述需求

| ❌ 不好的描述 | ✅ 好的描述 |
|------------|-----------|
| "做个网页" | "做一个个人网站，包含头像、介绍、联系方式" |
| "加个按钮" | "添加一个深蓝色的登录按钮，放在页面右上角" |
| "好看一点" | "使用渐变背景，圆角设计，阴影效果" |
| "修bug" | "点击提交后没有反应，控制台显示xxx错误" |

### 技巧2：如何分步骤完成任务

```
复杂任务分解：

任务：做一个待办事项应用

❌ 一次性描述：
你：帮我做一个待办事项应用

✅ 分步骤描述：

你：首先，帮我创建一个待办事项列表的界面
→ AI创建界面

你：现在添加一个输入框，可以输入待办事项
→ AI添加输入框

你：添加一个按钮，点击后把输入的内容添加到列表
→ AI添加按钮和功能

你：让列表中的每一项都可以点击删除
→ AI添加删除功能

完成！
```

### 技巧3：如何调整AI的输出

```
不满意AI的输出时：

你：这个按钮太大太丑了
AI：好的，我来调整...

更好的方式：
你：按钮改成：
- 大小：中等（比如padding 10px 20px）
- 颜色：蓝色
- 圆角：5px
- 阴影：轻微阴影

AI：好的，按照你的要求调整按钮...
（AI按照精确的规格修改）
```

## 📚 代码基础知识

### HTML是什么？

HTML（超文本标记语言）是网页的"骨架"。

```
HTML的作用：定义网页的内容

示例：
<h1>这是标题</h1>
<p>这是段落</p>
<img src="图片.jpg">
<button>这是按钮</button>
```

### CSS是什么？

CSS（层叠样式表）是网页的"皮肤"。

```
CSS的作用：定义网页的样式

示例：
h1 {
    color: red;        # 文字颜色
    font-size: 20px;   # 字体大小
    text-align: center; # 居中对齐
}

button {
    background: blue;   # 背景颜色
    padding: 10px 20px; # 内边距
    border-radius: 5px;  # 圆角
}
```

### JavaScript是什么？

JavaScript是网页的"肌肉"。

```
JavaScript的作用：让网页能动起来

示例：
function sayHello() {
    alert("你好！");
}

// 点击按钮时调用
button.onclick = sayHello;
```

## 🐛 常见初学者问题

### Q1: AI生成的代码不能用

```
可能的原因：
1. 没有保存文件
2. 文件扩展名不对
3. 缺少依赖

解决方法：
1. 确保文件已保存
2. 检查文件名是否正确
3. 让AI检查并修复
```

### Q2: 不知道怎么开始

```
解决方法：
1. 从最简单的Hello World开始
2. 按照本章的示例一步步来
3. 不懂就问AI
4. 多做几次就熟悉了
```

### Q3: AI的回复不是我想要的

```
解决方法：
1. 更清楚地描述你的需求
2. 分步骤，一次只要求一个功能
3. 给出具体的要求（颜色、大小等）
4. 如果还是不对，让AI重新生成
```

### Q4: 代码看不懂

```
解决方法：
1. 选中不懂的代码，让AI解释
2. 让AI用更简单的语言解释
3. 让AI给例子说明
4. 多问几次，慢慢理解
```
